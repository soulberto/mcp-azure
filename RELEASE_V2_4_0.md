## 🚀 Instalación

```bash
npm install @slorenzot/mcp-azure@2.4.0
```

## 🔄 Migración desde v2.3.x

Esta versión es **100% compatible** con v2.3.x. No se requieren cambios en la configuración existente.

Simplemente actualiza la versión:

```bash
npm update @slorenzot/mcp-azure
```
